# Questcast -- GitHub Issue Labels

## Bug Severity

| Label | Color | Description |
|-------|-------|-------------|
| `bug/p0-critical` | `#d73a4a` (red) | App crash, data loss, security breach, voice loop broken |
| `bug/p1-high` | `#e36209` (orange) | Feature broken, workaround exists |
| `bug/p2-medium` | `#fbca04` (yellow) | Cosmetic issue, minor UX problem |
| `bug/p3-low` | `#0e8a16` (green) | Nice-to-have improvement |

## Type

| Label | Color | Description |
|-------|-------|-------------|
| `feature` | `#1d76db` (blue) | New feature or capability |
| `enhancement` | `#5319e7` (purple) | Improvement to existing feature |
| `documentation` | `#0075ca` (light blue) | Documentation only |
| `refactor` | `#d4c5f9` (lavender) | Code refactoring, no behavior change |
| `performance` | `#f9d0c4` (peach) | Performance improvement |
| `security` | `#b60205` (dark red) | Security-related issue |
