## 💡 Product Summary

### What is Questcast?
Questcast is a **voice-first mobile RPG** that uses AI to create immersive tabletop gaming experiences. Players speak their actions, and an AI Dungeon Master responds with dramatic narration and real-time generated illustrations.

### Key Features
- 🎙️ **Voice-First Interface** - Speak your actions, hear the story
- 🤖 **AI Dungeon Master** - Adaptive storytelling powered by GPT-4o-mini
- 🎨 **Real-Time Illustrations** - AI-generated scene images
- 👥 **Multiplayer Support** - 1-6 players per session
- 🌍 **Multilingual** - Czech, English, German (and more)

### Monetization
- **Freemium Model**: Same quality, limited quantity
- **Subscriptions**: $9.99 - $24.99/month
- **In-App Purchases**: Session extensions, extra content
- **Adventure Pass**: Seasonal content ($4.99/month)
